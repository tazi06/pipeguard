# Security Policy

## Supported Versions

We release security updates for the latest major version of PipeGuard. Please always use the latest release for best security.

| Version | Supported          |
| ------- | ----------------- |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in PipeGuard, **please do not open a public issue**. Instead, report it privately:

- Email: hakkacheyassine@gmail.com | https://www.linkedin.com/in/hakkache-yassine-857362160/
- Or use GitHub's [private vulnerability reporting](https://github.com/tazi06/pipeguard/security/advisories/new)

We will respond as quickly as possible and coordinate a fix and disclosure.

## Security Best Practices

- Always use the latest version of PipeGuard
- Review scan results for critical/high issues
- Never run untrusted pipelines or files
- Keep your system and dependencies up to date
